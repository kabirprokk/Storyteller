## 🔧 Customization

Easy to modify:

1. **Change Colors**: Edit CSS variables in `:root`
   ```css
   :root {
       --primary: #3b82f6;  /* Change to your color */
   }
   ```

2. **Add More Posts**: Edit mock data in `loadMockData()`
   ```javascript
   const mockPosts = [ /* add more */ ]
   ```

3. **Change Content**: Update any HTML text directly

4. **Add Features**: Add JavaScript functions to the script section
